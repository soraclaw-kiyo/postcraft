## OpenClawとは？24時間働くAIエージェントフレームワーク

**OpenClaw**は、2025年末に「Clawdbot」として誕生し、2026年1月にリブランドされた**無料・オープンソースのAIエージェントフレームワーク**です。

従来のChatGPTやClaudeのような「チャットボット」とは根本的に異なります。チャットボットは「聞かれたら答える」受動的なツールですが、OpenClawは**自律的にタスクを実行する能動的なAIエージェント**です。

### OpenClawの特徴を一言で言うと

> あなたのパソコンやサーバーで24時間365日動き続ける「AIの秘書チーム」

具体的には以下のことができます：

- **メッセージアプリ連携**: Telegram、Discord、Slack、WhatsApp、iMessageに直接接続
- **システムレベルのアクセス**: ファイル操作、ブラウザ制御、メール処理、カレンダー管理
- **マルチエージェント**: 複数のAIエージェントがチームとして協力して作業
- **永続メモリ**: 会話の文脈を覚えていて、長期的に学習・改善
- **スキルシステム**: プラグインのように機能を追加できるモジュール構造

### 従来のAIツールとの違い

| 項目 | ChatGPT/Claude | OpenClaw |
|------|---------------|----------|
| 動作方式 | Webブラウザで手動操作 | バックグラウンドで自律動作 |
| 稼働時間 | 使う時だけ | 24時間365日 |
| 外部連携 | 限定的 | Telegram、Discord等と直接連携 |
| ファイル操作 | アップロードのみ | ローカルファイルを直接読み書き |
| マルチエージェント | 非対応 | 複数エージェントが協力可能 |
| コスト | 月額$20〜 | 無料（LLM APIキー代のみ） |

## OpenClawのセットアップ方法【初心者向け・15分で完了】

### 必要なもの

1. **パソコン**（Mac、Linux推奨。Windowsも対応）
2. **Node.js**（JavaScriptの実行環境。無料）
3. **LLM APIキー**（Anthropic Claude、OpenAI GPT、Google Geminiなど）
4. **メッセージアプリのボットトークン**（Telegram推奨）

### ステップ1: Node.jsのインストール

Node.jsの公式サイト（nodejs.org）からインストーラーをダウンロードして実行します。バージョン18以上が必要です。

### ステップ2: OpenClawのインストール

ターミナル（コマンドプロンプト）を開いて、以下のコマンドを実行します：

```
npm install -g @anthropic/openclaw
```

これで自動的にOpenClawがインストールされます。

### ステップ3: ワークスペースの初期化

```
openclaw init
```

このコマンドで、設定ファイルが`~/.openclaw`フォルダに作成されます。

### ステップ4: LLM APIキーの設定

Anthropic Claudeを使う場合：

```
openclaw models auth paste-token --provider anthropic
```

APIキーを入力すると設定完了です。APIキーはAnthropic公式サイトのコンソールから取得できます（初回$5分の無料クレジット付き）。

### ステップ5: エージェントの作成

**SOUL.md**というファイルでエージェントの性格や役割を定義します。これがOpenClawのユニークな点で、Markdownファイル1つでAIの振る舞いを完全にカスタマイズできます。

### ステップ6: Gatewayの起動

```
openclaw gateway start
```

これでOpenClawが起動し、バックグラウンドで動き始めます。Web UIは`localhost:18789`でアクセスできます。

### ステップ7: メッセージアプリと接続

TelegramのBotFatherでボットを作成し、トークンをOpenClawに設定すれば、Telegramからいつでもエージェントに話しかけられます。

## OpenClawのスキルシステムとは

OpenClawの大きな特徴の一つが**スキルシステム**です。スキルとは、エージェントに特定の能力を追加するプラグインのようなものです。

### 主なスキル例

- **Web検索スキル**: インターネットから最新情報を取得
- **ブラウザ操作スキル**: Webサイトを自動操作
- **画像生成スキル**: AIによる画像作成
- **音声合成スキル**: テキストから自然な音声を生成
- **GitHub連携スキル**: リポジトリ管理やPR作成を自動化

スキルはMarkdownファイル（SKILL.md）で定義されるため、プログラミング知識がなくても作成・カスタマイズが可能です。

## マルチエージェント機能で「AIチーム」を構築

OpenClawの最も強力な機能が**マルチエージェント**です。ACP（Agent Communication Protocol）を使って、複数のエージェントが連携して動きます。

### チーム構成の例

| エージェント名 | 役割 | 使用モデル |
|-------------|------|----------|
| Sora（ボス） | 指揮・判断 | Claude Opus |
| Sou（開発） | コーディング | Claude Opus |
| Shin（リサーチ） | 調査・分析 | Claude Sonnet |
| Hibiki（ライター） | 記事執筆 | Claude Sonnet |

各エージェントが専門分野を持ち、ボスエージェントが仕事を割り振る。まさに**AIの組織**です。

## よくある質問（FAQ）

### Q1: プログラミング経験がなくてもOpenClawは使えますか？

**はい、基本的な操作は可能です。** セットアップにはコマンドラインの基本操作（コピー&ペースト程度）が必要ですが、オンボーディングウィザードが手順を案内してくれます。エージェントの設定はMarkdownファイルの編集なので、プログラミングの知識は不要です。

### Q2: OpenClawを動かすのにどれくらいのスペックのPCが必要ですか？

**一般的なノートPCで十分動作します。** OpenClaw自体は軽量なNode.jsアプリケーションです。推奨スペックはメモリ4GB以上、ストレージ1GB以上。ただし、ローカルLLMを動かす場合は別途GPUが必要になります。

### Q3: OpenClawは安全ですか？データが外部に漏れることはありませんか？

**データはすべてローカルに保存されます。** OpenClawはセルフホスト型のため、設定ファイルやメモリデータは自分のマシン上にあります。LLM APIへの送信データのみが外部通信となりますが、これは通常のChatGPT利用と同レベルです。2026年のアップデートではサンドボックス実行やシークレット管理など、セキュリティがさらに強化されています。

### Q4: OpenClawとChatGPTは何が違いますか？

**最大の違いは「自律性」です。** ChatGPTは質問に答えるチャットボットですが、OpenClawはバックグラウンドで自律的にタスクを実行するエージェントです。ファイル操作、メッセージアプリ連携、スケジュール実行など、ChatGPTにはできないシステムレベルの操作が可能です。

### Q5: 複数のLLMを同時に使い分けることはできますか？

**はい、可能です。** OpenClawはモデル非依存設計のため、タスクの重要度やコストに応じてClaude Opus（高性能）、Sonnet（コスパ）、Gemini Flash（高速・低コスト）などを使い分けられます。
